# The Butterfly Effect / Variabel X

**[Sensitive Dependence on Initial Conditions]**

Selamat datang di titik nol.
Halaman ini bukan sekadar kumpulan kode, melainkan jejak digital dari sebuah upaya kecil yang diharapkan memicu dampak non-linear di masa depan.

**Tujuan:**
Mengeksplorasi batas antara "Kemustahilan" (takdir statis) dan "Kemungkinan" (takdir dinamis). Apa yang Anda lihat di sini adalah bukti bahwa perubahan besar bisa dimulai dari perangkat paling sederhana.

* **Origin:** Indonesia
* **Device:** Mobile / Low-End
* **Mission:** Disrupting the equilibrium.

> "A single spark can start a prairie fire."
